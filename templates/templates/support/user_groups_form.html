{% extends 'GenericForm.html' %} {% block formheader %} A user cannot have more than two groups. In most cases, one group is sufficient.

The following group types are available

Name Rights
supervisors 'Responsible Staff' for a project. Can also be assistant to a project, like assistant.
unverified Unverified user. This is what new users automatically become on first login. The user can later be verified by a support user. It is not possible to assign this group to a user.
assistants 'Assistant' for a project, the daily supervisor.
studyadvisors Support user, makes the distributions and is able to make do all kind of management tasks. Be careful, this group gives the users a lot of power!
directors Directors can view all projects
groupadministrator Has read and/or write access to all projects of specific group, has to be specified in group administrator form
